

## LogoFlow: Attention-Guided Adversarial Logo Generation against Siamese Phishing Detectors

The repository contains code and datasets to generate adversarial logos that bypass Visual Phishing Detectors (i.e., PhishIntention). 


## Environment
```bash
conda create -n LogoFlow python=3.8
conda activate LogoFlow
```

```bash
pip install -r requirements.txt
```

## Install

```bash
$ pip install rectified-flow-pytorch
```

## Training

```python
python train_logo.py
```
## Sampling

```python
python sampling.py
```
